
# Imunisasi Gamification

An interactive web app for exploring immunization and health information, developed for dr. Cempaka DF - Puskesmas Cimandala.

## Features

- **Clickable Hotspots:** Clickable areas over a classroom background, each revealing information or media in a modal popup.
- **Modal Info Display:** Clickable areas open a modal with images or videos related to the selected object.
- **Background Music:** Ambient music plays automatically (with user interaction fallback) and can be toggled.
- **Responsive Layout:** Works on desktop and mobile browsers.
